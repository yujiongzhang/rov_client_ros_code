# rov_client ros qt
## DEPENDENCE
```
ubuntu 22.04
ros2 humble
qt 5.15.3
QMake 3.1
```
## FILE DESCRIPTION

dir | note
---------|----------
include | header file
src     | source file
resource | resource file such as pictures
ui | ui for qt , designed by QT5 Designer
install | self defined message types

## HOW TO START
```
source install/setup.bash
mkdir build && cd build
cmake ..
make -j 8
```
